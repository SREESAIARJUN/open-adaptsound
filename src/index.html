Open-AdaptSound

Personalize your sound

A short listening check tunes your headphones to how you hear — then applies it system-wide with zero lag.

Checking Equalizer APO…
Requires Equalizer APO

This app does not process audio itself. It configures Equalizer APO (free system audio driver). First-time setup:

  1. Install Equalizer APO (official installer)
  2. In Device Selector, enable your headphones
  3. Reboot Windows when asked
  4. Open Open-AdaptSound again → run the beep test → Apply

We don’t bundle the driver inside this EXE: APO needs device selection + a reboot, which can’t be fully automated safely.

  • 🎧 Wear your headphones and keep them on the whole time.
  • 🔔 Beeps only · Yes / No · 15 full-range bands per ear (to 16 kHz).
  • 🤫 Quiet room · ~5 minutes · soft beeps only.
Required driver

Equalizer APO setup

Open-AdaptSound writes your hearing profile into Equalizer APO. Without that driver, profiles can’t go system-wide.

1. Download & install Run the official Equalizer APO installer (admin prompt is normal)
2. Device Selector Check the box for your headphones / output device → OK
3. Reboot Windows APO attaches to the audio stack — a restart is required after install
4. Come back here Open Open-AdaptSound → beep Yes/No test → Apply Adapt Sound Profile

Why not inside our installer? APO’s Device Selector needs you to pick devices, silent install still needs that step, and Windows expects a reboot. Shipping the driver inside our EXE would be fragile and confusing — we launch the official setup instead.
