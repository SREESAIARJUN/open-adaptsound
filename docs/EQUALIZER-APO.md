# Equalizer APO

Open-AdaptSound only **writes config**. Playback EQ is done by [Equalizer APO](https://sourceforge.net/projects/equalizerapo/).

## Setup

1. Install Equalizer APO  
2. Device Selector → enable **headphones**  
3. **Reboot**  
4. Open Open-AdaptSound → beep test → Apply  

Download: https://sourceforge.net/projects/equalizerapo/files/latest/download

## Why not bundled?

- Device selection is interactive  
- First install expects a reboot  
- Official updates stay on SourceForge  

After APO is set up, profile changes hot-reload (no reboot).
