# Brand mark

**Open-AdaptSound logo meaning**

- **Blue left wave + violet right wave** — independent left/right hearing
- **Asymmetric shapes** — ears are not identical; the profile adapts per side
- **Center focus ring + mint node** — calibrated threshold (the softest level you still hear)
- **Not classic EQ bars** — this is hearing personalization, not a mixer graphic
