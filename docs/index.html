Why this exists

Windows plays one generic mix. It does not know your ears, your headphones, or which bands you hear less well. Music-player EQs stay in one app. Manual Equalizer APO is powerful but easy to mis-set.

Without it

  • Factory sound for a “generic” listener
  • EQ only inside Spotify / one player
  • Guessing which bands to boost
  • Same curve for both ears

With Open-AdaptSound

  • Matched to how you hear
  • System-wide via Equalizer APO
  • Beep Yes/No finds thresholds
  • Independent left / right peaks

How it works

  1. Hearing check — pure sine beeps, left then right, 15 bands to 16 kHz. Answer Yes or No.
  2. Profile — thresholds become dB gains + preamp headroom (Gentle / Balanced / Full).
  3. Apply — writes Equalizer APO config (hot-reload, keeps your other filters).
  4. Use — every app is equalized at the Windows audio stack. This app is not in the playback path.

headphones → beep test → Open-AdaptSound → Equalizer APO → all apps

Install

1. Equalizer APO

  1. Download Equalizer APO
  2. Device Selector → enable headphones
  3. Reboot when asked

2. Open-AdaptSound

  1. Download the .exe installer
  2. Quiet room → beep test (Y/N)
  3. Apply profile

We don’t bundle APO: device selection and first reboot can’t be fully automated. Why.

FAQ

Is this a medical hearing test?

No. Practical headphone check for EQ—not a clinical audiogram.

Do I need Equalizer APO?

Yes for system-wide results. This app only writes the profile.

Will it add game latency?

After Apply, EQ runs in Equalizer APO (low latency). Open-AdaptSound is not processing live audio.

Headphones or speakers?

Designed for headphones so left/right tones stay isolated.

Does it wipe my existing APO config?

No. It adds an Include file for the personal profile.
